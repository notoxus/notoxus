### About Me

I'm a security-focused builder who believes the best way to defend a system is to first learn how to break it. I spend my time turning complex security concepts into hands-on practice — and into content other people can actually learn from.

- **Offensive mindset, defensive goal** — sharpening my practical pentesting skills on TryHackMe and hunting for new vulnerabilities.
- **"Systematically Lazy"** — if I do something twice, I script it. I build small automation tools to streamline workflows (e.g. a script that bundles a whole project — classes, READMEs and all — into a single file for fast, AI-assisted code reviews).
- **I learn by teaching** — I break down technical topics into structured blogs and vlogs, because explaining something is the real proof that I understand it.
- **Open to collaborate** on security projects, automation scripts, and knowledge-sharing initiatives.
